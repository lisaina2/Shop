# Shop

## Краткое описание

Shop — это интернет‑магазин, разработанный на фреймворке Django. Проект предоставляет базовый функционал для электронной торговли: каталог товаров, систему управления заказами, административную панель для управления контентом, а также механизмы аутентификации и авторизации пользователей.

## Инструкции по установке

Чтобы развернуть проект на локальной машине, выполните следующие шаги:

1. Скопируйте репозиторий:
```bash
git clone <URL_РЕПОЗИТОРИЯ>
```
*(замените `<URL_РЕПОЗИТОРИЯ>` на фактический URL вашего репозитория)*

2. Перейдите в директорию проекта:
```bash
cd myshop
```

3. Создайте виртуальное окружение Python:
```bash
python -m venv venv
```

4. Активируйте виртуальное окружение:
- Для Linux/macOS:
  ```bash
  source venv/bin/activate
  ```
- Для Windows:
  ```cmd
  venv\Scripts\activate
  ```

5. Установите зависимости из файла `requirements.txt`:
```bash
pip install -r requirements.txt
```

6. Выполните миграции базы данных:
```bash
python manage.py migrate
```

7. Создайте суперпользователя (администратора) для доступа к панели управления:
```bash
python manage.py createsuperuser
```
*(система запросит имя пользователя, email и пароль — введите нужные данные)*

8. Запустите сервер разработки:
```bash
python manage.py runserver
```

## Запуск проекта

После выполнения всех шагов проект будет доступен по адресу:  
http://127.0.0.1:8000

Админ‑панель Django доступна по адресу:  
http://127.0.0.1:8000/admin

## Требования

- Python 3.8+
- Django 4.0+
- Список зависимостей указан в файле `requirements.txt`

## Структура проекта

- `myshop/` — корневая директория проекта
- `manage.py` — утилита для управления проектом
- `requirements.txt` — файл со списком зависимостей
- `venv/` — виртуальное окружение (создаётся при установке)
- `<название_приложения>/` — директории приложений Django
